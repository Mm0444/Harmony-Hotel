# 🏨 Harmony Hotel — Data Analytics Project

> **CS4473** | Hotel Business Data Analysis & Dashboard Design

---

## 📋 Project Overview

This project explores the data analytics process within the hotel industry. Using modern data analytics tools and techniques, we analyze hotel operational data to extract meaningful insights and support informed business decisions — presented through an intuitive, easy-to-understand dashboard built in **Power BI**.

---

## 🎯 Objectives

- Study and apply data analytics processes relevant to the hotel business using industry-standard tools and technologies.
- Transform analyzed data into actionable insights by designing and building a **Dashboard** that is clear, visual, and decision-ready.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **Power BI** | Dashboard design & data visualization |
| **Microsoft Excel** | Data preprocessing & cleaning |

## ⚙️ How to Use

### 1. Preprocess the Data
Open the files in the `Preprocess data/` folder — data has already been cleaned and is ready for use.

### 2. Open the Dashboard
Open the `.pbix` file in **Power BI Desktop** to explore the interactive dashboard.

> 💡 If you don't have Power BI Desktop, download it free from [Microsoft](https://powerbi.microsoft.com/desktop/).
